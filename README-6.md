# CareerFlow — Job Application Tracker

**Live Site:** [asma-umar-dev.github.io/Career-Flow](https://asma-umar-dev.github.io/Career-Flow/)

CareerFlow is a premium job application tracker that helps you organize, monitor, and analyze every step of your job search in one elegant workspace.

## ✨ Features

- **Dashboard** — Clear daily overview of your job search:
  - Daily goal tracker (e.g., 0/2 applications)
  - Weekly target tracker (e.g., 0/5 applications)
  - Recent Activity feed
  - Upcoming events (interviews, deadlines) via Calendar
- **Applications** — Full application tracking with a detailed form:
  - Company name, job title, location
  - Work type (Remote / Hybrid / On-site)
  - Job type (Full-time / Part-time / Internship / Contract)
  - Salary range
  - Status pipeline: Wishlist → Applied → Under Review → Assessment → Interview Scheduled → Final Interview → Offer Received → Accepted / Rejected
  - Application date, interview date, deadline
  - Job posting link and personal notes
  - Edit, duplicate, or delete any application (with confirmation on delete)
- **Analytics** — Insights into your job search performance over time
- **Calendar** — Visual view of interviews and deadlines
- **Settings** — Customize workspace preferences
- **Dark Mode** — Toggle between light and dark themes

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (single-page static app)
- Local data persistence (browser-based storage)
- Hosted via **GitHub Pages**
- Responsive, sidebar-navigation workspace UI

## 📁 Getting Started

Clone the repo and open `index.html` directly, or serve it locally:

```bash
git clone https://github.com/asma-umar-dev/Career-Flow.git
cd Career-Flow
# then open index.html in your browser
# or serve locally:
npx serve .
```
